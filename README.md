# Ecommerce Landing Page

## Description

This is a responsive ecommerce landing page built using Tailwind CSS components as part of my training program.
The project focuses on modern UI design, responsiveness, and utility-first CSS styling.

# Installation & Setup

 1. Clone the repository:
   
   git clone https://github.com/raoamitkumar01/tailwind-project-1/tree/main

2. Open the project folder:
   
   cd tailwind-project-1
3. Open index.html in your browser

## Features
- Responsive design (mobile, tablet, desktop)

- Clean and modern UI

- Product showcase section

- Hero section with call-to-action

- Navigation bar and footer

- Reusable Tailwind components


# Tech Stack

- HTML5
- Tailwind CSS

# What I Learned

- How to use Tailwind CSS utility classes effectively
- Building responsive layouts
- Structuring a landing page
- Using pre-built Tailwind components
